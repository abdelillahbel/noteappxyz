## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/abdelillahbel/noteappxyz.git
    ```
2. Navigate to the project directory:
    ```sh
    cd noteappxyz
    ```
3. Install dependencies:
    ```sh
    flutter pub get
    ```
4. Run the app:
    ```sh
    flutter run
    ```
